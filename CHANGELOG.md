## [1.0.0] - 2026-01-28

- 初始版本发布
- 后端 API 服务（SpringBoot 4 + JDK 25）
- 管理后台前端（Vue3 + Element Plus）

- H5 端前端（Vue3 + Vant）
- Docker 容器化部署支持
- 基础的用户、部门、课程管理功能
- 统一的响应结果封装
- 跨域配置
- 数据库初始化脚本

- 后端：SpringBoot 4, JDK 25, MyBatis Plus, MySQL, Redis
- 前端：Vue 3, TypeScript, Vite, Element Plus, Vant
- 部署：Docker, Docker Compose, Nginx
